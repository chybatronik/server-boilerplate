# Orionsoft GraphQL Server Boilerplate

Installation

```sh
$ git clone https://github.com/orionsoft/server-boilerplate [name of your app]
$ cd [name of your app]
$ rm -rf .git
$ yarn
$ meteor npm install --save bcrypt
$ meteor
```
